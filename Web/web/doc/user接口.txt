#获取用户信息--------------------------------------------------------------------
/user/get

访问:POST

参数:
aim
args

aim=all&args=all 获取全部user
aim=id&args=具体Id 精确查询

返回
精确查询
status
message
user{
    id
    username
    lasttime
    motto
    id
    profession
    college
    registertime
    grade
    email
    permit 中文
    permitInt 数字
}

全部查找
status
message
size list中有多少个user包
list{
    0:{第一个user的信息}
    1:...
}

#修改密码接口--------------------------------------------------------------------
/login/changePassword

访问POST

参数

正常更改密码

aim=normal
password
newpassword
code 普通验证码


忘记密码

aim=forget
newpassword
code 普通验证码

返回

status
message

#管理员改变权限---------------------------------------
/user/admin/changePermit

访问POST

参数
aim up|down|equal
userid
permit 当aim为equal时需要

权限范围 0-9 开放测试时登录后可随意升降

返回
status
message
